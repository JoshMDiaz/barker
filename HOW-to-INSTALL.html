Getting started

1.Installation

cd into the directory and then run npm install command.

$ npm install

2.Configuration

Before run ionic serve

1.Create a new firebase project

2.Open the Database section. Click DATA tab > Import JSON > firebase_DATA.json

3.Enable Email/Password sign-in

    -Open the Authentication section.

    -On the SIGN-IN METHOD tab, enable the Email/password sign-in method and click Save.

4.Open the Database section. Click RULES tab

******* insert this code *******

{
  "rules": {
    ".read": "true",
   ".write": "auth != null"
  }
}

5.Update Firebase configuration in /src/app/app.module.ts

********** firebase configuration ***********

export const config = {
   apiKey: your-key,
   authDomain: your-project-authdomain,
   databaseURL: your-database-URL,
   projectId: your-project-id,
   storageBucket: your-storage-bucket,
   messagingSenderId: your-messaging-sender-id
};

3.Run

Testing your app in a browser is as simple as running the serve command in your project’s root folder.

$ ionic serve


Facebook sign-in setting

STEP 1 :

First enable facebook Sign-In in firebase console

Go into firebase console > Authentication > Sign-In Method > Facebook


Click the Enable toggle.


Now you need to enter your Facebook App ID and App secret.



STEP 2 :

To get Facebook App ID and App secret.

You first have to create a new Facebook App inside of the Facebook developer portal at https://developers.facebook.com/apps.

Click “Add a new App”

Once you click on the button, you’ll get a short form pop up, add the name you want for your app and the contact email that will be public to users.

App ID and App secret can be found in your Facebook app under settings:

Now copy facebook App ID and App Secret to firebase console



STEP 3 :

Add your Platforms to Facebook

Once everything is set up in our development environment, we need to let Facebook know which platforms we’ll be using (if it’s just web, iOS, or Android).

In our case, we want to add Android platforms.

To add the platforms, go ahead and inside your Facebook dashboard click on settings, then, right below the app’s information you’ll see a button that says Add Platform, click it.

Click Android

Enter your Google Play Package Name

To get Google Play Package Name

Open config.xml and copy package name

Paste into Google Play Package Name



STEP 4 :

Adding the Cordova Facebook Plugin to your app

Now you are able to add the Facebook plugin:

$ ionic cordova plugin add cordova-plugin-facebook4 --variable APP_ID="123456789" --variable APP_NAME="myApplication" --save.

You’ll need to replace the values or APP_ID and APP_NAME for your real credentials. You can find both of those inside your Facebook Developers Dashboard.

So the next thing we need to do is install the facebook package from Ionic Native, open your terminal again and type:

$ npm install --save @ionic-native/facebook




Google sign-in setting

STEP 1 :

First enable google Sign-In in firebase console

Go into firebase console > Authentication > Sign-In Method > Google


Click the Enable toggle and save.


Now you need to reopen google Sign-in again.


Click Project Settings.


Click Add Firebase to your Android app.

Once you click on the button, you’ll get a short form pop up, now add Android Package Name.

To get Android Package Name

Open config.xml and copy package name

Then paste it into Android Package Name

Now you need to add debug signing certificate SHA-1

To get Debug signing certificate SHA-1 key

Run this command:

keytool -exportcert -list -v -alias androiddebugkey -keystore ~/.android/debug.keystore

For window user:

Please view this video

Now copy SHA-1 from command line

Then paste it into Debug signing certificate SHA-1 and click REGISTER APP



STEP 2 :

Add Cordova Plugin

cordova plugin add cordova-plugin-googleplus --save --variable REVERSED_CLIENT_ID=myreversedclientid

Note: myreversedclientid in above command should be replaced by WEB Client Id in Reverse and it will start like com.googleusercontent.apps.XXXXXXXXXXXX

To get WEB Client Id

Go into firebase console > Authentication > Sign-In Method > Google > click Web SDK configuration

INSTALL GOOGLE PLUS NATIVE RAPPER

$ npm install --save @ionic-native/google-plus


STEP 3 :

Replace Web Client Id in providers/auth_data.ts file

<button ion-button color="bluegray" >Button</button>

Theme

-Copy variables.scss (src/theme/variables.scss) to your src/theme folder

-Usage : color="color name"

EXAMPLE :

<ion-navbar color="lightblue" ></ion-navbar>
<button ion-button color="bluegray" >Button</button>

Google map

Add the Google Maps JavaScript SDK

- Include the Google Maps API in your src/index.html file

- Replace Google Maps API key (To generate an API key, go to this page and follow the instructions.)

<script src="http://maps.google.com/maps/api/js?key=YOUR_API_KEY_HERE"></script>

Add the Geolocation plugin

$ npm install --save @ionic-native/geolocation

Animation

-Copy animation.scss (src/theme/animation.scss) to your src/theme folder

-Import '../theme/animations' in your src/app/app.scss file

@import '../theme/animations';

-Usage : class="effect name"

EXAMPLE :

<button ion-button class="slide-up" >Button</button>
